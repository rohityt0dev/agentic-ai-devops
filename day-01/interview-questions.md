# 🦙 Q2 — What is Ollama?

> Ollama is a local LLM runtime used to run Large Language Models (LLMs) offline on your machine.


## 📦 Supported Models

- Llama 3
- Mistral
- Gemma
- DeepSeek
- Phi

## ⚡ Why Developers Use Ollama

- Faster local testing
- Better privacy
- No API cost
- Easy experimentation
